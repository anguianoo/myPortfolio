Quiz App

I created this project to challenge my jquery skills. This project is a simple quiz app I made through Thinkful. It was to be 5 questions long, and have a score/question section.
I click on the box that displays the correct answer. Based on the selection, the quiz will tell me if the answer chosen is correct or not.

Tech stack: HTML, CSS, and JavaScript
